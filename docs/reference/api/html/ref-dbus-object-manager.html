The /org/freedesktop/ModemManager1 object

org.freedesktop.ModemManager1 — The ModemManager Manager interface.

The ModemManager process will export an object at the well-known path /org/freedesktop/ModemManager1.

This object, which implements the standard org.freedesktop.DBus.ObjectManager, is responsible for managing the list of Modem objects.

This object also controls any process-wide operation, such as the log level being used by the daemon.
